# 個人網頁 - Ting Hao

這是一個為 **Ting Hao** 設計的現代化個人單頁網站，採用了最新的 **Glassmorphism (玻璃擬態)** 設計風格，並結合了動態時鐘功能。
Live Demo:https://hao20030419.github.io/NCHU_DIC_HW1/
## ✨ 特色功能

- **即時時鐘 (Live Clock)**：
    - 每秒自動更新，精確顯示小時、分鐘與秒數。
    - 數字採用 `tabluar-nums` 等寬字型設定，避免數字跳動。

- **視覺設計 (Design)**：
    - **玻璃擬態卡片**：採用磨砂玻璃質感 (Frosted Glass)，搭配背景模糊 (Backdrop Blur) 與精細邊框。
    - **動態背景**：深紫色與靛藍色的漸層背景，隨時間緩慢流動。
    - **漂浮光球**：背景配置了三個不同顏色與動態的模糊光球，營造視覺深度。
    - **流光文字**：名字 "Ting Hao" 具有紫色漸層流光動畫效果。

- **使用者體驗 (UX)**：
    - **進場動畫**：所有元素載入時皆有平滑的淡入上浮 (Fade-up) 效果。
    - **互動效果**：卡片在滑鼠懸停時會有輕微的上浮與陰影加深效果。
    - **響應式設計**：支援手機與桌面版面自動調整。

## 🛠️ 技術細節

- **HTML5**：語意化標籤結構。
- **CSS3**：
    - 使用 CSS Variables 管理色票。
    - 運用 `@keyframes` 製作漸層流動、光球漂浮與文字流光動畫。
    - 使用 Flexbox 進行置中排版。
- **JavaScript**：
    - 原生 JS 實作時間與日期邏輯。
    - 使用 `setInterval` 實現每秒即時更新。
- **字體**：整合 Google Fonts (Outfit & Inter) 呈現現代感。
